
# Day 08 - Viewing and Editing Files with Nano

## Objective

The goal for today was to learn how to open, edit, and save files using the nano text editor in Linux.

---

## What I Learned

- I learned how to open files using `nano`
- I understood how to save files using `CTRL + O`
- I learned how to exit the editor using `CTRL + X`
- I learned how to view file contents using `cat`

---

## What I Built / Practiced

- Created text files and edited them using nano
- Practiced saving and exiting files
- Viewed file contents using `cat`

---

## Challenges Faced

- Initially unfamiliar with nano shortcuts
- Needed practice to remember how to save and exit

---

## Key Takeaways

- Nano is a simple and useful text editor in Linux
- Editing files directly is better than using echo for longer text
- Keyboard shortcuts are important when using terminal editors
