
# Day 17 - File Processing in Linux

## Objective

The goal for today was to learn how to process files using shell scripting and extract useful data.

---

## What I Learned

- Files can be read line by line using `while read`
- Data can be split using `IFS`
- Text files can be processed like datasets
- `grep` can filter data
- `wc` can count lines in a file

---

## What I Built / Practiced

- Created a sample dataset file
- Wrote scripts to read file content
- Extracted fields like name and age
- Filtered data using conditions
- Counted number of records

---

## Challenges Faced

- Understanding how to split data using `IFS`
- Writing conditions inside loops

---

## Key Takeaways

- Shell scripting can process structured data
- Linux tools can simulate basic data pipelines
- File processing is essential in data engineering
- Combining loops and conditions makes scripts powerful

---

## Resources

- Linux manual pages (`man grep`, `man wc`)
- Practice on remote Linux server
