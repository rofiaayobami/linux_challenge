  # Day 12 - Linux Networking Fundamentals

## Objective

The goal for today was to understand basic networking concepts and learn how to test and interact with network connections in Linux.

---

## What I Learned

- I learned that every system has an IP address used for communication
- I understood the difference between IPv4 and IPv6
- I learned that `ip` is the modern tool for network configuration
- I learned how to test connectivity using `ping`
- I learned how to fetch data using `curl`
- I learned how to download files using `wget`

---

## What I Built / Practiced

- Checked my IP address using `ip a`
- Tested network connectivity using `ping`
- Retrieved web content using `curl`
- Downloaded files using `wget`
- Viewed routing information using `ip route`

---

## Challenges Faced

- Understanding networking concepts at first
- Differentiating between various networking tools

---

## Key Takeaways

- Linux provides powerful networking tools
- `ip` is the preferred modern command over `ifconfig`
- `ping` helps test connectivity
- `curl` and `wget` are used to interact with web resources
- Networking knowledge is essential for system administration
