
# Day 05 - File Compression and Archiving

## Objective

The goal for today was to understand how to combine multiple files into a single archive and compress them to reduce file size, as well as how to extract them back when needed.

---

## What I Learned

- I learned the difference between archiving and compression in Linux
- I understood how to use `tar` to combine multiple files into one archive
- I learned how to compress and decompress files using `gzip` and `gunzip`
- I learned how to extract compressed archives using `tar -xvzf`

## What I Built / Practiced

- Created files and added content using `echo`
- Used `grep`, `find`, `wc`, `head`, and `tail` to review file content
- Compressed and decompressed files using `gzip` and `gunzip`
- Created both `.tar` and `.tar.gz` archives and extracted them

---

## Challenges Faced

- Made a mistake with file naming when trying to extract an archive
- Initially confused about the difference between archiving and compression
- Had to practice the `tar` command options to understand them properly

---

## Key Takeaways

- Archiving combines multiple files into one file
- Compression reduces the size of files
- `.tar.gz` is commonly used because it combines both archiving and compression
- Linux commands are sensitive to spelling, so accuracy is important
