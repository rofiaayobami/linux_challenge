## What I Learned

- i learned that linux is based on UNIX operating system (OS), UNIX being an operating system built in the 1970s
- i that linux has over 600+ distributors each of which are designed for different purposes. e.g UBUNTU is recommended for begineers, linux mint is easier to navigate for window users etc
- i learnt some of its importance with emphasis on its level of high security and stability, free and open source and large global community support
- i learnt that linux is made up of 5 components which includes the KERNEL - core of linux OS, SYSTEM LIBRARIES - allows applications interact with kernal with no need to access it directly, SHELL - the command line interface (CLI) that allowas communication with the OS, HARDWARE - physical components and SOFTWARE UTILITIES - built in tools to help users.
- i learnt to use ls - to list my directory, cd- change directory, pwd - print working directory

## Key Takeaways

- mkdir - to make directory 
- ls - to view folders and files in a directory
- ls -la - for a more comprehensive listing
- cd - change directory
- pwd - print working directory
- touch - to create files
- cat - view file content 
- echo - to insert into a file 
- rm - to delete or remove a file 
- cp - copy a file 
- mv - to move or rename a file
