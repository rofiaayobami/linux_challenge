
# Day 09 - Linux File System

## Objective

The goal for today was to understand how the Linux file system is structured and how files and directories are organized within the system.

---

## What I Learned

- I learned that all files and directories in Linux start from the root directory `/`
- I understood that the Linux file system follows a hierarchical (tree-like) structure
- I learned the purpose of important directories like `/home`, `/etc`, `/bin`, and `/var`
- I gained basic understanding of how Linux manages files internally

## Key Takeaways

- Linux organizes files in a tree structure starting from `/`
- Each directory has a specific purpose in the system
- Understanding the file system makes navigation easier
- Not all details need to be memorized at once
