
# Day 06 - Package Management

## Objective

The goal for today was to understand how to install, update, and remove software packages in Linux using the apt package manager.

---

## What I Learned

- I learned how to update the package list using `apt update`
- I understood how to install software using `apt install`
- I learned how to remove packages using `apt remove`
- I learned how to search for available packages using `apt search`

---

## What I Built / Practiced

- Updated the system package list
- Installed a package (`tree`) and used it
- Searched for available packages
- Removed installed packages

---

## Challenges Faced

- Limited permissions prevented some of the `sudo` commands from working
- Needed to understand the difference between update and upgrade
- Some commands required administrative privileges

---

## Key Takeaways

- `apt` is used to manage software in Linux
- Updating the package list is important before installing software
- `sudo` is required for system-level changes
- Not all users have permission to install packages
