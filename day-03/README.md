# Day 03 - [Topic]

## Objective

What was the goal for today?

To learn how to search for files and find specific information within files using Linux commands.

---

## What I Learned

-  I learned how to search for files using the `find` command
- I understood how to search for specific words inside files using `grep`
- I learned how to use `wc`, `head`, and `tail` to analyze file content
- I understood the concept of piping (`|`) to combine commands


---

## What I Built / Practiced

- Created multiple text files and added content using `echo`
- Searched for files using `find . -name "filename"`
- Used `grep` to find specific words in files
- Practiced combining commands like `cat` and `grep`


---

## Challenges Faced

- Initially made mistakes with command syntax (e.g., incorrect use of `find`)
- Entered multi-line mode accidentally due to incomplete commands
- Confusion between overwriting (`>`) and appending (`>>`) in files


---

## Key Takeaways

- The `find` command is used to locate files in directories
- The `grep` command is useful for searching text within files
- Proper command syntax is very important in Linux
- Using `Ctrl + C` helps to exit stuck or incomplete commands
