
# Day 13 - Disk Usage in Linux

## Objective

The goal for today was to understand how to monitor disk space and identify which files or directories are consuming storage.

---

## What I Learned

- I learned how to check total disk space using `df`which means disk free
- I learned how to check directory sizes using `du`which means disk usage
- I learned how to display results in human-readable format using `-h`
- I learned how to summarize directory size using `du -sh`
- I learned how to sort directories by size

---

## What I Built / Practiced

- Checked system disk space using `df -h`
- Checked folder sizes using `du -sh`
- Analyzed directory usage using `du --max-depth=1`
- Sorted directories by size to identify large files

---

## Challenges Faced

- Understanding the difference between `df` and `du`
- Interpreting the output of disk usage commands

---

## Key Takeaways

- `df` shows overall disk usage
- `du` shows size of files and directories
- `-h` makes output easier to read
- Sorting helps identify large files quickly
- Disk monitoring is important for system performance
